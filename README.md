# IT2 - Mitt første API med flask
 
## Slik kjører du flask på PC-en din

> OBS! Ikke bruk utvidelsen `flask-vgs`

Skriv følgende i terminalen:


### Mac

1. `python3 -m venv venv`
2. `. venv/bin/activate`
3. `pip install -r requirements.txt`
3. `flask run`

### Windows

1. `py -3 -m venv venv`
2. `venv\Scripts\activate`
3. `pip install -r requirements.txt`
4. `flask run`
